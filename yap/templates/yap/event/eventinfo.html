Videochat Platform Selection:

  • If you select Use Jitsi Meet (free), YapSpot will create the room for you and email the URL link to all attendees automatically. No further work is required. (Jitsi Meet is an excellent free-to-use videochat platform. Link: https://meet.jit.si/)
  • If you want to use Zoom, Google Meet, Skype, or another service, then select Use other videochat platform (e.g. Zoom, Google Meet, Skype, etc.). You may enter the URL for the event below and information for finding the event below. If you don't have the URL ahead of time, then return to the event page at least 15 minutes prior to the start of the event so you can add the information for finding the meetup in the event description. This will be automatically emailed to all attendees.
